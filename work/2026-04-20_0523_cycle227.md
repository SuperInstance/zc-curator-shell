- **docs‑research** – Documentation, research notes, literature reviews, memory logs
- **experimental** – Proof‑of‑concept, one‑off tests, sandbox projects
- **abandoned** – Inactive, deprecated, or superseded projects

**Tags (for finer granularity):**
- `training` – Involves model training, LoRA adapters, data generation
- `simulation` – Simulated environments, synthetic data generation
- `runtime` – Execution environment, VM, kernel
- `geometry` – Spatial math, constraint solving, CAD‑like operations
- `edge` – Jetson, embedded, low‑power deployment
- `cloud` – Cloud services, lighthouse, orchestration
- `protocol` – Communication standards, APIs
- `research` – Exploratory, non‑production
- `tool` – Utility, configuration, benchmarking
- `archive` – Historical, inactive but kept for reference
