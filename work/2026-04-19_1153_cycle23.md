- **plato** – PLATO room system, training infrastructure, LoRA export

- **docs** – Documentation, protocol definitions, memory files
- **tools** – Utilities, scripts, service management, health checks
- **research** – Exploratory code, rabbit‑trail investigations, concept proofs
- **legacy** – Old prototypes, deprecated experiments, historical artifacts
- **staging** – In‑progress work, unmerged branches, experimental forks

### Tags (for cross‑category filtering)
- **rust** – Written in Rust
- **python** – Written in Python
- **torch** – Uses PyTorch
- **cuda** – Uses CUDA/GPU kernels
- **simulation** – Agent or environment simulation
- **training** – Model training or data generation
- **server** – Network service or daemon
- **cli** – Command‑line tool
- **visual** – Graphics, UI, or visualization
- **archive** – Read‑only, frozen state
