- `gpu` – GPU‑accelerated or CUDA‑based
- `rust` – Written in Rust
- `python` – Written in Python
- `typescript` – Written in TypeScript/JavaScript
- `api` – Provides or consumes an API
- `cli` – Command‑line tool
- `web` – Web interface or server
- `experimental` – High‑risk, unstable, or proof‑of‑concept
- `stable` – Production‑ready or actively maintained
- `archive` – No longer actively developed
